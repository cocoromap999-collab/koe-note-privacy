よくある質問FAQ

  • 文字起こしが表示されないTranscription not showing
    設定 → プライバシー → 音声認識 で「Koe Note」をオンにしてください。 Go to Settings → Privacy → Speech Recognition and enable Koe Note.
  • 録音できないCannot record
    設定 → プライバシー → マイク で「Koe Note」をオンにしてください。 Go to Settings → Privacy → Microphone and enable Koe Note.
    • AIチャットが動かないAI Chat not working
    設定画面でAnthropicのAPIキーが正しく入力されているか確認してください。 Check that your Anthropic API key is correctly entered in the app Settings.
  • 購入の復元ができないCannot restore purchase
    設定画面 →「購入を復元」をタップしてください。同じApple IDでログインしていることを確認してください。 Tap "Restore Purchase" in Settings. Make sure you're signed in with the same Apple ID used for the original purchase.
  • ウィジェットが表示されないWidget not showing
    ホーム画面を長押し → ＋ → 「Koe Note」を検索して追加してください。 Long press the home screen → tap + → search "Koe Note" and add the widget.
  • データを削除したいDelete my data
    設定画面 →「すべてのデータを削除」からいつでも削除できます。 Go to Settings → "Delete All Data" to remove all your data at any time.
  • 返金を希望したいI want a refund
    当社（Nagi Studio）による返金対応は行っておりません。返金が必要な場合は、Apple サポート（reportaproblem.apple.com）へお問い合わせください。 Nagi Studio does not provide refunds directly. Please contact Apple Support (reportaproblem.apple.com) for refund requests.
